Ejercicio 1: Configuración de Videojuego
El escenario: Estás guardando las preferencias de un jugador para un juego de disparos. Necesitamos guardar lo siguiente en un solo archivo:
El nombre del jugador es "Slayer99".
El volumen de la música está al 75 por ciento.
El modo de dificultad elegido es "Veterano".
La opción de "Invertir Eje Y" (para mirar arriba/abajo) está activada.
No tiene ningún clan asignado (está vacío).
Piensa: ¿Qué lleva comillas? ¿Qué es número? ¿Cómo representas "activado" o "vacío"?

Ejercicio 2: Catálogo de API (El Array Raíz)
El escenario: Un servidor te pide que le envíes una lista de precios de criptomonedas. No quiere metadatos ni configuraciones, solo la lista pura y dura para procesarla rápido. Tienes dos monedas para enviar:
Bitcoin: vale 45000.50 dólares y su símbolo es BTC.
Ethereum: vale 3200 dólares y su símbolo es ETH.
Piensa: ¿Cómo empieza el archivo si es solo una lista? ¿Cómo manejas los decimales?
